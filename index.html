Domain

Literature Survey & Research Context

Clinical, technical, and social foundations underpinning the CareSync platform for elderly community care.

Ageing Population Challenges

Sri Lanka's rapidly ageing population faces significant gaps in community-based elderly care. Social isolation, undetected emotional decline, and limited caregiver resources create critical unmet needs. Traditional in-person monitoring is unsustainable at scale, necessitating AI-assisted remote care solutions.

Affective Computing & Emotion Recognition

CNN-based facial emotion recognition (FER) using models trained on FER2013 and AffectNet datasets achieves real-time inference via OpenCV and TensorFlow/Keras. 7-class emotion classifiers with grayscale 48×48 preprocessing and Haar Cascade face detection enable scalable, low-latency detection suitable for continuous monitoring.

Natural Language Understanding — Low-Resource Languages

Sinhala remains severely under-resourced in NLU research. Rasa-based intent/entity extraction pipelines adapted for Sinhala (si-LK) using browser Web Speech API + custom NLU training data represent a novel contribution. Existing elder care voice assistants exclusively target high-resource languages, leaving Sinhala-speaking populations unserved.

IoT-Based Fall Detection

Falls are the leading cause of fatal and non-fatal injuries in adults over 65. ESP32 + MPU6050 6-axis inertial measurement enables real-time, hardware-level fall detection with a 4-stage algorithm. MQTT-based cloud transmission via HiveMQ provides low-latency alerting. Existing elderly care systems lack integrated IoT fall detection — leaving a critical physical safety gap for unsupervised elders at home.

Research Gap

What's Missing

No existing elderly care platform integrates real-time CNN-based emotion detection, a Sinhala-language voice controller, IoT hardware fall detection, and a comprehensive caregiver analytics dashboard in a single unified ecosystem.Current solutions treat these as isolated components: emotion systems lack voice interfaces, voice assistants do not support Sinhala, fall detection requires proprietary hardware, and caregiver dashboards are not AI-informed. The Sri Lankan elderly care context is entirely unaddressed in existing literature.

Our Solution

CareSync

A fully integrated MERN-stack platform combining a Python CNN webcam emotion monitor, a Rasa Sinhala NLU voice controller with React frontend orchestration, an ESP32 + MPU6050 IoT fall detection system with MQTT cloud alerting, and an intelligent caregiver dashboard with mood trend analytics and fall history — all purpose-built for Sri Lankan elderly care contexts. The platform addresses language, hardware, and clinical gaps simultaneously in a single deployable system.

Objectives

Research Objectives

Four integrated AI subsystems addressing the wellbeing, communication, and monitoring needs of Sri Lankan elders.

Real-Time Emotion Monitoring

Deploy a pre-trained CNN (TensorFlow/Keras, 7-class FER) via OpenCV Haar Cascade face detection with live webcam feeds. Output emotion labels, confidence scores, and timestamps as JSON to a Node.js REST API for persistence and dashboard broadcast using dual REST + Socket.IO channels.

Sinhala Voice Controller

Build a full Sinhala NLU pipeline using browser Web Speech API (si-LK) → Rasa 3.6 intent/entity extraction → React frontend command orchestration. Support navigation, reminder CRUD, emergency alerts, and status queries — entirely in the Sinhala language with MongoDB persistence.

Caregiver Intelligence Dashboard

Design a React dashboard with 8 internal pages, real-time Recharts/Chart.js mood visualizations, mood trend analytics across days and weeks, and a frosted-glass sticky navigation. Connect to Node.js/Express backend with 20+ controller functions and 5 Mongoose schemas for comprehensive elder profiles.

IoT Fall Detection System

Deploy an ESP32 microcontroller with MPU6050 (6-axis accelerometer + gyroscope) to continuously monitor body movements. Implement a 4-stage detection algorithm — sudden movement, rotation, impact, and post-fall stillness — to minimize false alarms. Publish sensor data via MQTT (HiveMQ Cloud) to a Node.js backend, persist to MongoDB Atlas, and display real-time acceleration graphs and fall alerts on a React caregiver dashboard.

Solution

Solution Architecture

A four-layer AI platform — perception, understanding, management, and analytics — built on a MERN microservices foundation.

Emotion Detection Engine (Python/CNN)

Python cnnapp.py — OpenCV VideoCapture → Haar Cascade face detection → 48×48 grayscale crop → normalize [0,1] → TensorFlow/Keras .h5 model inference → color-coded bounding box + 7-class probability bars. Live FPS counter and togglable stats overlay. Outputs JSON packets via REST to Express.js logging endpoint.

Sinhala NLU Voice System (Rasa + React)

Browser SpeechRecognition (si-LK) → VoiceButton.jsx → Flask /parse API → Rasa Agent (model.gz) → intent + entities. VoiceController.js routes: UI commands execute client-side (navigate, scroll); reminder intents POST to voiceRoutes.js → VoiceReminderController.js → MongoDB Reminder schema. Sinhala success feedback shown to user.

Node.js Backend & Data Layer

Express.js server with dual channels: REST API for auth, bulk logging, and CRUD operations; Socket.IO for real-time emotion broadcast to dashboard. MongoDB Atlas with 5 Mongoose schemas — Emotion Logs, Caregiver Profiles, Reminders, Alerts, and Elder Records. 20+ controller functions for complete platform management.

IoT Fall Detection (ESP32 + MQTT)

ESP32 microcontroller reads MPU6050 6-axis accelerometer + gyroscope data continuously. A 4-stage algorithm (sudden movement → rotation → impact → post-fall stillness) classifies fall events to reduce false alarms. On detection, the ESP32 publishes a JSON payload to HiveMQ Cloud via secure MQTT over Wi-Fi. Node.js subscribes, validates, and persists to MongoDB Atlas. React dashboard renders real-time Chart.js acceleration graphs and live fall alert cards for caregivers.
